6. 國科會計畫 - 建立新世代個人化雲端健康記錄醫療資訊服務系統 (大五，2013)：

建立醫療雲端服務，記錄每個人的健康記錄，希望除了未來可供個人健康記錄跨醫療單位調閱查詢 (尤其是影像報告調閱)，以及個人健康記錄管理外，也可提供政府、學術單位進行人口統計、疾病統計、用藥統計等應用，或許能減少很多國民健康保險的開銷。

我們依據 Integrating the Healthcare Enterprise (IHE) Cross Enterprise Document Sharing (XDS) 作為交換基礎架構，使系統交換機制、文件內容標準化，讓系統有擴充性、再製性，以減少不同系統介面間不相容而導致的成本。

主要以 Java 作為開發語言，使用到 Web Service SOAP 作為交換機制，以 ZK 作為網頁開發介面，更透過 Apache Maven 管理專案、函式庫。

我們分別於 2013年 3 月、9月至北京天壇醫院參與聯測 (IHE-C Connectathon 2013)，大會招集各個廠商在會期內互相連接系統，在有限時間內更改系統，互相測試，通常前三天壓力會非常大，每個人絞盡腦汁，尋找不通過的原因；然而，通過數百項使用案例後，即可獲得證書證明，而具有證明的系統才可導入至醫療機構。

7. 輔導記錄登錄系統 (大學畢業後，暑假)：

提供一個平台，讓各個教師在輔導學生後，登錄輔導記錄，因各處室所需輔導單張不同，需要統整並將收集到的記錄內容，提供教師評鑑系統作為評鑑依據。使用 asp.net MVC 專案開發，其具備成熟的 MVC架構，並有 Object Relational Mapping (ORM) 機制，讓系統與資料庫相依性降低，不會因系統隨需求切換不同的資料庫時，而做出重大修改；MVC 架構是 RESTful API 實作的典範，能提供 API 並搭配 Jquery AJAX 功能存取資料，減少重新讀取非必要地資訊。

8. 碩士研究論文 -自動擷取心臟內科病患病史資訊 (碩士)：

電子醫療記錄有豐富的醫療資訊，但病歷通常以自由文本記錄，難以被電腦有效的利用，而資訊擷取技術能將重要的資訊明確地被表達，若是將病歷結構化呈現則利於資料分析，或是有相關機構的要求，要求從現有病歷中擷取出需要的格式以便於通報等後續應用。

自然語言處理技術，已被應用作為非結構自由文本的識別和資訊擷取，相較於人類自行檢視醫療記錄，資訊擷取技術更有效率以及一致性，本研究以心臟內科入院病歷為例，探討機器學習技術或依據人類知識歸納的規則式作為基礎，能自動擷取病患病史資訊。

使用到 Spring MVC、Spring Security + Thymeleaf，作為網頁開發基礎，自然語言處理核心目前嚐試使用 Apache cTAKES、CRF++、The Stanford NLP等開源碼工具，並解決在病歷上自然語言處理的困難問題，拼錯字修正、縮寫分歧、句子邊界偵測、否定詞分辨等，在統整醫生所興趣的分類後，說明在大量的病歷之中有此疾病的人有哪些人、發生的時間、是病人發生的疾病還是家屬發生的疾病，可幫助醫師收集大量資料作回溯性研究。

三、社團、運動、休閒：
