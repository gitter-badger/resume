Gaduo's Resume

關於 Gaduo：

一、 弄巧成巧

        進入醫學資訊學系，因為高中生物非常不錯，又因系上是新成立而且成績符合所以推甄前來，沒想到進來，程式課程比重高，生物幾乎是蛋白質、DNA 等議題，漸漸的我對於程式有些興趣，當時加入排球代表隊，因為練球時間與晚上的程式規畫課程上機有衝突，為了能有充分時間練球，而每當上機題目一公布後，我用最努力的方式將作業完成，我沒有特別聰明，所以勤能補拙，常常坐上8小時左右的電腦，為了是完成作業去練球，而老師將題目分成 10 個等級求最好，我幾乎是完成最高等級，這樣就可以安心的去練球了，已奠定往後的程式扎實的基礎。

二、 學長領進門，修行在個人

        大二下，當時還流行 BBS 留言板，已畢業的學長在版上詢問是否有人有意願開發 MSN 機器人，對於一個剛接觸程式的我非常有興趣，於是常常跟學長交流，雖然當時沒有完成 MSN 機器人，但我比同儕還更早接觸到網頁以及 Android，Android 小專案像是：翻譯年糕，將語音辨識搭配Google翻譯以及文字轉語音輸出 (TTS)，提供使用者用說話的方式翻譯成不同語言並播放; 動態心電圖，將事前讀取好的心電圖訊號能動態的繪畫在手機螢幕上。
        同時間，我也在練習 PHP 網頁開發，從完全沒有網頁開發概念，看書自學或與計算中心大哥交流，到想如何模組化快速開發，最後遵照 MVC 架構；當時室友們一回寢室即是開團打魔獸，我則是在開發商務網站，形成強烈的對比，回憶起來還滿有趣。

三、 校園生活

        在大學期間參加的社團只有排球代表隊以及排球社，除了每周兩天的練習外，偶爾會到附近國小教導學生排球，遇上運動會時，也會協助裁判，所謂運動志工，而系際盃時擔任裁判，裁判工作不簡單，需要強烈的專注力，更要代表公正的力量，不偏不移的決策每顆球。
        計算機中心工讀生，在此學習組裝電腦、軟體安裝、備分外，更重要的是如何排除過濾問題的能力，印象最深刻的是大三當時宿舍網路設備老舊，又P2P 軟體橫行，每當門禁過後，宿舍網路即是大斷網，那一兩個月常常需要奔波於男女宿之間，深夜才能回寢室，雖然辛苦也造福了很多人；單位氣氛，大哥們很照顧我們，平時可以交流到很多經驗，每個期末甚至還有聚餐，說是慰勞大家的辛勞；同仁間感情也不錯，週末還會一同相約單車同遊。

四、 手工藝

        對於 Coding 這件事情來說，不只是想學好程式語言，而是希望做好每一件作品，我其實很熱愛作手工藝，像是十字繡、木工、串珠、拼布，可說是能文能武，所以每當有作品可以展現，我都希望用挑剔的態度將作品完成，回到 Coding 除了將系統完成，更重要的是追求如何模組化，讓維護成本降低，讓程式之間耦合度降低，提高內聚力是我在這方面常常給自己的挑戰，所以在 Java Spring 哲學上是我很希望學習好，也還在努力之中的一個架構。
