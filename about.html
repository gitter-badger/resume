Gaduo's Resume

關於 Gaduo：

一、 弄巧成巧

        大學就讀慈濟大學醫學資訊學系，對生物相當自信的我，原以為此科系所學是以「生物」為重，沒想到「程式」才是主菜，而生物學相關的，只有蛋白質、DNA等議題，但意料之外的，我開始對程式產生了興趣。
        大學的生活中，一年級我加入了排球代表隊，碰巧的是練球時間與程式規劃的實習課衝突，所幸老師都會提前公佈實習課題目，為了能練球又能兼顧學業，每當實習題目一公佈，便是竭盡所能地將題目完成，當時老師將題目分成10個等級，我幾乎是完成最高等級。我沒有特別聰明，只能以勤來補拙，當時一天在電腦前8小時幾乎是常態，但卻為我的程式能力奠下扎實的基礎。

二、 學長領進門，修行在個人

        大二時還流行著BBS，已畢業的學長在版上詢問，是否有人有意願開發MSN機器人，對於剛接觸程式的我，感到相當有興趣，就像拿到入場卷一樣興奮，當時雖然沒有完成MSN機器人，但藉由與學長的交流，我比同儕還更早接觸到網頁以及Android。
        當時我也練習PHP網頁開發，從完全沒有概念開始，看書自學，也與計算機中心的大哥交流，如今想的問題是，如何模組化、快速開發，直到最後遵照 MVC 架構；回憶當時，室友們回寢室便是開團打魔獸，我則是在開發商務網站，強烈的對比還蠻有趣的。
        在大學中完成了Android 小專案，像是「翻譯蒟蒻」，將語音辨識搭配Google翻譯，以及文字轉語音輸出(TTS)，提供使用者用說話的方式，翻譯成不同語言並播放；「心電圖」，將事前讀取好的心電圖訊號，動態繪製在手機螢幕上。

三、 校園生活

        在大學期間所參加的社團有排球代表隊及排球社，除了每週兩天的練習外，偶爾還會到附近國小教排球，遇上學校運動會時，便協助擔任運動志工，而系際盃時擔任裁判，然而裁判工作不簡單，需要強烈的專注力，更要代表公正的力量，不偏不移的決策每顆球。
        因為家境因素，大學的生活費，都要靠自己一點一滴籌措，當時在計算機中心擔任工讀生，學習組裝電腦、軟體安裝及備分外，更重要的是學到了如何排除及過濾問題的能力；印象最深刻的是，大三當時宿舍網路設備老舊，又P2P軟體橫行，每當門禁過後，宿舍網路即是大斷網，因此常常要於男女宿舍間奔波，直至深夜才能回寢室休息，雖然辛苦，卻也造福了很多人。

四、 手工藝

        我其實很熱愛作手工藝，像是十字繡、串珠、拼布等，可說是能文能武，每當有作品可以展現，我都是用挑剔的態度將作品完成。
        在 Coding 上，我不只是想學好程式語言，同樣的，是希望做好每一件「作品」，除了將系統完成外，更重要的是如何模組化，讓維護成本下降，使程式之間耦合度降低，而提高內聚力，我在這方面常常自我挑戰，所以在 Java Spring哲學，是我希望學習好，也是正在努力的一個架構。
